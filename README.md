# Kamkar - Job Matching Platform

A simple web-based platform built with HTML, CSS, and jQuery to connect employers and job seekers.

## 🚀 Features
- Employer and job seeker registration / login
- Job posting and browsing
- Profile creation for both parties
- Basic matching system using jQuery interactions

## 💻 Technologies Used
- **HTML5** – For structuring the content
- **CSS3** – For styling and responsive layout
- **jQuery** – For dynamic user interactions

├── css/ # Stylesheets
